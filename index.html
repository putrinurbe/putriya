About Me

I’m a student majoring in Network and Computer Systems with a strong passion for databases and web development. I love exploring new technologies and solving problems through coding and system optimization.

Saya telah mengerjakan berbagai proyek di berbagai bidang, termasuk pengembangan web, IoT, desain grafis, dan pembuatan konten. Saya telah membangun beberapa situs web sederhana, dengan fokus pada pengalaman pengguna dan desain responsif. Selain itu, saya mengembangkan proyek IoT untuk penghangat makanan gorengan, yang mengintegrasikan perangkat keras dan perangkat lunak untuk meningkatkan efisiensi. Selain proyek teknis, saya memiliki pengalaman dalam desain grafis, membuat konten visual untuk platform digital. Saat ini, saya sedang mendalami manajemen basis data untuk memperdalam pemahaman saya tentang penyimpanan data, pengorganisasian, dan pengoptimalan. Saya selalu bersemangat untuk mempelajari teknologi baru dan meningkatkan keahlian saya.
